# AI Agent Guidelines

This repository contains baseline instructions for AI agents that build web
products, APIs, and supporting infrastructure.

All agent instructions are consolidated in one file:

```text
AGENTS.md
```

Projects that install these guidelines should keep the instructions directly in
their root `AGENTS.md`. Do not add a `.ai-guidelines/` directory.

## Setup

Quick install from the root of a target repository:

```bash
curl -fsSL <install-script-url> | sh
```

For manual setup options, read `setup.md`.
